# API Testing Dashboard

Test and debug REST APIs using curl and jq.

## Features

- Send HTTP requests (GET, POST, PUT, DELETE)
- View formatted JSON responses
- Save and replay test cases
- Export results

## Development

This is a static single-page application. Simply open `index.html` in your browser or deploy to GitHub Pages.

## Deployment

GitHub Pages is configured via `.github/workflows/deploy.yml`. After enabling Pages in repository settings, the site will be available at:

```
https://personalbotai.github.io/api-testing-dashboard/
```
